### Installation
Just grab `DPad.js` from this repo.

### Usage
Init the script with your buttons.
```js
Touche.init({"btn1": {
    "press": function () {
        //presed
    },
    "unpress": function () {
        //unpresed
    },
    "style":{
        "left":"50%",
        "top":"50%"
    }
}});
```
